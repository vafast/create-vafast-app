# My Vafast App

使用 [Vafast](https://github.com/vafast/vafast) 创建的高性能 TypeScript API。

## 开始使用

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

访问 http://localhost:3000

## 可用脚本

- `npm run dev` - 启动开发服务器（热重载）
- `npm run start` - 启动服务器
- `npm run build` - 编译 TypeScript
- `npm run serve` - 运行编译后的代码
